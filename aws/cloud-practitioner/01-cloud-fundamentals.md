## What is Cloud Computing

- Is the on-demand delivery of IT resources, particularly compute power application hosting, database application, networking, and more

## Cloud deployment models

- **Cloud** - All in cloud
- **On-Premises** - Not in cloud
- **Hibryd** - Some in cloud and some not in cloud

## Cloud service models

- **IaaS** — Infrastructure as a Service
- **PaaS** — Platform as a Service
- **SaaS** — Software as a Service

## Benefits of Cloud

- **Trade upfront expense for variable expense**- **Upfront Expenses (CAPEX)** - Large upfront investment in hardware before use. **Variable expenses (OPEX)** - Pay for usage and request. Give back what you don't need
- **Stop focusing on data centers** - **Focus on customers** - Invest time and money; focus on customers and applications
- **Stop guessing capacity (or easily fix bad guesses)** - **Extreme scalability** - Scales in and out as needed; experiments with load and performance ad needed.
- **Benefit from massive economies of scale** - **Economies of scale** - Benefit from customers' aggregated usage, with price drops per unit used
